# Arkraft Confluence 문서 분석

> 분석일: 2026-01-13

## 현황

### 스페이스 분산 (SSOT 위반)

| 스페이스 | 경로 | 내용 |
|----------|------|------|
| ARK | `ARK > Arkraft` | ADR, Sprints, 회고, 개발 |
| AMT | `AMT 26년 운영 > Arkraft` | Agents, GTM, Status, Reference |

**문제**: 동일 프로젝트가 두 스페이스에 분산

### 페이지 폭발

| 유형 | 개수 | 기간 |
|------|------|------|
| 일일 싱크업 | 40+ | 2025.11~12 |
| 연구 일지 | 5+ | 2025.12 |
| 주간 롤업 | 4 | 11월 1~4주차 |

### 명명 불일치

```
2025.11.04 일일 싱크업      (정상)
25025.11.14-16 일일 싱크업  (오타)
2025.11.28 - 11.30          (형식 다름)
2026.01.06 큰거오나?        (비정형)
```

### 카테고리 부재

루트에 혼재된 문서 유형:
- 아키텍처 분석
- 개발 이슈
- 기획 문서
- 리서치

## 권장 조치

### 1. SSOT 확립

ARK 스페이스를 SSOT로 지정, AMT는 링크로 참조

### 2. 폴더 구조

```
Arkraft/
├── Status (대시보드)
├── Sprints/
├── ADR/
├── Retrospectives/
└── Backlog/

회의록/
├── 2025-11/
└── 2025-12/

아키텍처/
리서치/
기획/
개발/
```

### 3. 명명 규칙

```
일일: YYYY-MM-DD 일일 싱크업
주간: YYYY-WXX 주간 롤업
ADR: ADR-XXXX: 제목
```

### 4. 정리 루틴

| 주기 | 작업 |
|------|------|
| 주간 | 일일 싱크업 → 주간 롤업 통합 |
| 월간 | 오래된 일일 페이지 아카이브 |
| 분기 | 폴더 구조 점검 |

## 참고

- ADR-0001: Git = Working SSOT (Confluence = Stakeholder View)
- Status 페이지: Jira/GitLab 연동 대시보드
