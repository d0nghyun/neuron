# Schedule Module

Notion 기반 일정관리 모듈.

## Features

- 오늘/주간 일정 조회
- 일정 추가/수정/완료
- Notion DB 연동

## Configuration

**Notion Database ID:** `2f346706-90ca-8108-b838-dd5861292951`

## Usage

neuron에서 모듈 활성화:
```
/workflow-init-module schedule
```

또는 직접 스킬 사용:
```
/workflow-schedule-notion
```

## Skills

| Skill | Description |
|-------|-------------|
| workflow-schedule-notion | 일정 조회, 추가, 상태 변경 |
